PBInfo 3182: marte2 - solutie

Cerinta

Pe planeta Marte anul marțian durează z zile marțiene. Marțianul Iggle își serbează ziua de naștere și constată că a trăit n zile. Câte lumânări va avea tortul lui Iggle?

Date de intrare

Programul citește de la tastatură numerele naturale z n.

Date de iesire

Programul va afișa pe ecran numărul de lumânări de pe tortul lui Iggle.

Restricții și precizări

1 ≤ z ≤ 1.000

1 ≤ n ≤ 1.000.000.000

Exemplu:

Intrare

7 56
Ieșire

8


Explicatie


Anul marțian are 7 zile. Iggle a trăit 56 de zile, adică 8 ani, deci tortul va avea 8 lumânări.

Solutie


#include <iostream>
using namespace std;
int main()
{
    int z,n, l;
    cin>>z>>n;
    l= n/z;
    cout<<l;
    return 0;
}
