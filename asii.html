1260: asii

Cerința

Se citesc două numere naturale. Afișați suma, diferenţa, produsul şi câtul lor, în această ordine, separate prin câte un spațiu.

Restricții și precizări

cele două numere citite vor fi mai mici decât 500;

din primul număr se va scădea al doilea;

primul număr se va împărţi la al doilea; se va afișa câtul împărțirii întregi.

Exemplu:

Intrare

15 3
Ieșire

18 12 45 5

#include <iostream>
using namespace std;
int main()
{
    int a,b,s,d,p,c;
    cin>>a>>b;
    s=a+b;
    d=a-b;
    p=a*b;
    c=a/b;
    cout<<s<<" "<<d<<" "<<p<<" "<<c<<" ";
    return 0;
}
