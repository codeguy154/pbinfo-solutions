PBInfo 1282: radical - solutie

Cerinta

Se dă un număr. Să se afișeze rădăcina sa pătrată.

Date de intrare

Programul citește de la tastatură numărul n.

Date de ieşire

Programul va afișa pe ecran numărul natural r, reprezentând rădăcina pătrată a lui n.

Restricţii şi precizări

1 ≤ n ≤ 1000

în fiecare test numărul n va fi pătrat perfect

Exemplu:

Intrare

9
Ieșire

3

Solutie


#include <iostream>
#include <cmath>
using namespace std;
int main()
{
    int n,r;
    cin>>n;
    r=sqrt(n);
    cout<<r;
    return 0;
}
