PBInfo 2377: pisici - solutie

Cerinta

Se cunoaște faptul ca n pisici mănâncă n șoareci în n minute. Sa se determine în cat timp mănâncă m pisici m șoareci.

Date de intrare

Programul citește de la tastatură numerele n si m.

Date de ieşire

Programul va afișa pe ecran numărul T, reprezentând timpul necesar pentru ca m pisici sa manance m soareci.

Restricţii şi precizări

1 ≤ n,m ≤ 100.000.000.000

Exemplu:

Intrare

3 100
Ieșire

3

Solutie


 #include <iostream>
using namespace std;
int main()
{
    long long int n,m,T;
    cin>>n>>m;
    T=m/m*n;
    cout<<T;
    return 0;
}
