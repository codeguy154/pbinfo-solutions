PBInfo 3210: cumparaturi - solutie

Cerinta

O cutie cu bomboane costă B lei. Gigel are S lei. Determinați câte cutii cu bomboane poate cumpăra Gigel și câți lei trebuie să mai primească pentru a cumpăra încă o cutie.

Date de intrare

Programul citește de la tastatură numerele naturale B S.

Date de iesire

Programul afișează pe ecran numerele C P, separate printr-un spațiu, reprezentând numărul de cutii cu bomboane pe care Gigel le poate cumpăra cu S lei, respectiv suma necesară pentru a cumpăra încă o cutie.

Restricții și precizări

1 ≤ B,S ≤ 1.000.000.000

Exemplu:

Intrare

4 9
Ieșire

2 3

Explicatie

Gigel are 9 lei, iar o cutie cu bomboane costă 4 lei. Poate cumpăra 2 cutii, rămânând 1 leu. Pentru încă o cutie, Gigel are nevoie de încă 3 lei.

Solutie


#include <iostream>
using namespace std;
int main()
{
    int b,s,C,P;
    cin>>b>>s;
    C=s/b;
    P=b-s%b;
    cout<<C<<' '<<P;
    return 0;
}
