PBInfo 815: lazi - solutie

Cerinta

Câte cutii cubice de latură l pot fi suprapuse într-o încăpere de înălțime h.

Date de intrare

Programul citește de la tastatură numerele naturale l și h.

Date de iesire

Programul va afișa pe ecran numărul n, reprezentând numărul de cutii ce pot fi suprapuse.

Restricții și precizări

1 ≤ l,h ≤ 1000000

Exemplu:

Intrare

2 7
Ieșire

3

Explicatie

Pot fi suprapuse doar 3 cutii, a patra nu mai intră.

Solutie


#include <iostream>
using namespace std;
int main()
{
    int l,h,n;
    cin>>l>>h;
    n=h/l;
    cout<<n;
    return 0;
}
