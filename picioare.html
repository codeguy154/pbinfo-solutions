PBInfo 175: picioare - solutie

Cerinta

Într-o curte sunt G găini și O oi. Să se determine numărul de capete și numărul de picioare din curte.

Date de intrare

Programul citește de la tastatură numerele G și O, reprezentând numărul de găini și numărul de oi din curte.

Date de iesire

Programul afișează pe ecran numerele C și P, separate printr-un spațiu, reprezentând numărul de capete și numărul de picioare din curte.

Restricții și precizări

0 < G , O < 1.000.000

Exemplu:

Intrare

3 2
Ieșire

5 14

Solutie


#include <iostream>
using namespace std;
int main()
{
    int G,O,C,P;
    cin>>G>>O;
    C=G+O;
    P=G*2+O*4;
    cout<<C<<" "<<P;
    return 0;
}
